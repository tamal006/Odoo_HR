An intelligent MERN stack application that combines AI capabilities with multiple MCP (Model Context Protocol) integrations to automate workflows, analyze data, and provide a seamless user experience.

## 🚀 Features

* 🤖 AI-powered assistant and automation
* 🔗 Multiple MCP integrations (Gmail, Google Drive, GitHub, and more)
* 📄 Document/PDF analysis
* 💬 Smart chatbot with contextual responses
* 🔐 Secure JWT-based authentication
* 📊 Modern responsive dashboard
* ⚡ RESTful APIs built with Express
* 🍃 MongoDB for scalable data storage

## 🛠️ Tech Stack

* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT
* **AI:** LLM-powered agent
* **Integrations:** MCP Servers (Gmail, Drive, GitHub, etc.)
